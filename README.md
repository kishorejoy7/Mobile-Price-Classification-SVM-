# 📱 Mobile Price Classification using SVM

This project uses a **Support Vector Machine (SVM)** to classify mobile phones into different price ranges based on their specifications.

## 🚀 Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## 📂 Dataset
- Mobile Price Classification Dataset
- Features: 20
- Target: `price_range`

## 📊 Model Workflow
- Load Dataset
- Data Preprocessing
- Feature Scaling
- Train-Test Split
- Train SVM Model
- Prediction
- Model Evaluation

## 📈 Evaluation
- Accuracy Score
- Confusion Matrix
- Classification Report
